Framework脚本打包并分发(针对framework依赖了其他包的基础上)

开发framework的时候我们会碰到这种情况，framework又依赖了其他的target,类似这样

也就是说，当我们想编译CYFrame的时候首先得编译OOO才可以，针对这样的情况我们来写一个脚本来自动进行打包，并发邮件。

脚本内容

大概流程是将OOO和CYFrame分别进行真机和模拟器结构打包，将framework里的.a文件进行合并，然后拷贝真机的framework到自己目录，拷贝的framework里的.a替换成合并后的.a，将制作完的framework压缩成zip，进行邮件发送，并输出当前framework的架构i386 x86_64 armv7 arm64,邮件发送借助的是sendEmail脚本
